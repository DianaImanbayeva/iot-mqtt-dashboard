# IoT Sensor Data Collection & Visualization

This project demonstrates a simple IoT data pipeline using a simulated sensor.

## Features
- Simulated IoT sensor (temperature and humidity)
- Data transmission using MQTT
- Data storage in SQLite
- Data visualization using Python (Matplotlib)

## Tools & Technologies
- Python
- MQTT (public broker)
- SQLite
- Google Colab

## Output
- Dashboard (time-series graph)
- GitHub repository
- Project report
